== Description ==
Supaz Easy Background - A simple plugin to add the background color or image to the specified div

Supaz Easy Background is a simple to use parallax background plugin that enables you to add background color, image or video to specified div in a unique way. It is best suited if you want to add parallax background to some divs to make it different from other sections. Image/Video will be automatically scaled with the inserted div, so regardless of the div size, the image will always fill the div, even as the browser is resized live. 

==Features==
* Easy configuration
* Supports image, video and background color.
* Parallax option for image and video background.
* Video supports - mp4 file, youtube or viemo.
* Various configurations for video support- video start time, video end time, video volumn mute, video volume level, play video only when visible.
* Disable background option when you don't need it.
* Option to disable parallax effect when you don't need it.
* Background overlay option to make your content more focused.
* Add as many as you want 
* And many more configrations.

== Installation =
1. Unzip supaz-easy-background.zip
2. Upload all the files to the /wp-content/plugins/supaz-easy-background
3. Activate the plugin through the \'Plugins\' menu in WordPress.
4. For customizing the plugin\'s settings, click on Supaz Easy Backgrounds option menu in WordPress left admin menu.
